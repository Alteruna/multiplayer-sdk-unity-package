# How to add a package to Unity.

> [!IMPORTANT]
> Unity's package manager depends on [Git](https://git-scm.com/install/) in order to install packages from a git url.

1. In Unity, open `Window → Package Manager`
2. In the top left corner, in the `+` dropdown, press "Add package from git URL..."
3. Input `https://github.com/Alteruna/multiplayer-sdk-unity-package.git`
4. Press `Add`<br>If everything is done right, you should now see something like this:<br>
    <img width="799" height="351" alt="image" src="https://github.com/user-attachments/assets/48a6ba88-b943-4fcf-9ae0-c8b81293b8ed" />
5. Register your project by going under `Window → Alteruna → Register license`
6. Paste your project ID or key in the field.<br>
   > *If you do not have a v2 Alteruna Project, get one from [portal.mp2.alteruna.com](https://portal.mp2.alteruna.com/)*

> [!NOTE]
> The version displayed in the Package Manager looks different than the actual version.\
> For example, 2.0.1r45 will be displayed as 2.0.1045
